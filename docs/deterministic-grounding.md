# Deterministic Grounding

The foundation of the Credential Lens pipeline is deterministic grounding. Before any AI or Language Model is invoked, the system relies on hard, mathematical facts about the document's layout. 

When a document is uploaded, it is rasterized and passed through a rigorous OCR sweep using engines like Tesseract (for images) or PDF text layer extraction (for digital PDFs). This process generates an array of `EvidenceBlocks`.

## Why is this critical?
Large Language Models, even multimodal ones, are fundamentally probabilistic. They hallucinate structure and can misinterpret the visual location of text if not constrained. By establishing a deterministic baseline, we don't have to guess where the text is; we map the exact spatial coordinates (bounding boxes) of every single token on the page. 

This baseline serves as the absolute truth. Later down the pipeline, any semantic structure generated by the VLM must be cross-referenced against this deterministic baseline to guarantee its existence on the actual document. This transforms an unreliable AI extraction process into a mathematically auditable, production-grade pipeline.
